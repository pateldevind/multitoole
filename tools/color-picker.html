How to Use
  1. Use the color picker or enter a hex color code to select a color
  2. View the color in different formats (RGB, HSL, CMYK, Named Color)
  3. Copy any color format to your clipboard
  4. Adjust brightness and saturation to modify the color
  5. Generate color harmonies (complementary, triadic, analogous)
  6. Save colors to your palette for future use
Note: All color conversions and manipulations are done entirely in your browser.
