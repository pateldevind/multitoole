Accessibility Guidelines
WCAG 2.1 Contrast Requirements:
  • Normal text (≤18pt): Minimum contrast ratio of 4.5:1
  • Large text (>18pt): Minimum contrast ratio of 3:1
  • UI components and graphics: Minimum contrast ratio of 3:1
How to Use
  1. Select foreground and background colors using the color pickers
  2. Choose a color vision type to simulate
  3. Adjust the severity slider to see different levels of color deficiency
  4. View the original and simulated appearance side by side
  5. Check the contrast ratios and accessibility guidelines
Tip: Use this tool to ensure your color choices are accessible to all users, including those with color vision deficiencies.
